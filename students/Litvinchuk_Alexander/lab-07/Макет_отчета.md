## Ход выполнения работы

### 1. Write Model

**Агрегат:** Session

**Структура:**
- Write Model представлен агрегатом `Session`
- Данные хранятся в нормализованном виде
- Основные поля: `session_id`, `status`
- Изменение состояния выполняется только через методы агрегата
- Агрегат генерирует доменные события:
  - `SessionStarted`
  - `SessionCompleted`
  - `SessionFailed`

**Инварианты:**
- Сессия создаётся в допустимом начальном состоянии
- Завершение изменяет статус на `DONE`
- Ошибка изменяет статус на `FAILED`
- Все изменения состояния сопровождаются регистрацией событий

**Файл реализации:**
- `lab-07/cqrs/write_model/session.py`

---

### 2. Read Model

**Проекция:** `SessionView`

**Структура:**
- Read Model представлен проекцией `SessionView`
- Данные хранятся в денормализованном виде для быстрого чтения
- Основные поля: `session_id`, `status`
- Проекция обновляется обработчиком событий `SessionProjection`

**Описание:**
Read Model используется только для чтения данных и не содержит бизнес-логики.  
В отличие от Write Model, проекция хранит уже подготовленное состояние, удобное для запросов.


### 3. Event-Driven Sync

**События:**
- `SessionStarted` → создать или обновить `SessionView` со статусом `ACTIVE`
- `SessionCompleted` → обновить `SessionView` со статусом `DONE`
- `SessionFailed` → обновить `SessionView` со статусом `FAILED`

**Код:**
```python
from sqlalchemy import create_engine, text
from sqlalchemy.orm import sessionmaker


class SessionProjectionHandler:
    def __init__(self, database_url: str):
        self.engine = create_engine(database_url, echo=True)
        self.SessionLocal = sessionmaker(bind=self.engine)

    def handle(self, event_type: str, session_id: str):
        db = self.SessionLocal()
        try:
            if event_type == "SessionStarted":
                db.execute(
                    text("""
                        INSERT INTO session_view (session_id, status)
                        VALUES (:session_id, :status)
                        ON CONFLICT (session_id)
                        DO UPDATE SET status = EXCLUDED.status
                    """),
                    {"session_id": session_id, "status": "ACTIVE"}
                )

            elif event_type == "SessionCompleted":
                db.execute(
                    text("""
                        UPDATE session_view
                        SET status = :status
                        WHERE session_id = :session_id
                    """),
                    {"session_id": session_id, "status": "DONE"}
                )

            elif event_type == "SessionFailed":
                db.execute(
                    text("""
                        UPDATE session_view
                        SET status = :status
                        WHERE session_id = :session_id
                    """),
                    {"session_id": session_id, "status": "FAILED"}
                )

            db.commit()
        finally:
            db.close()
```
